# Sky-Blue-Calculator

SkyBlue Calculator is a simple web-based calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and features a clean sky-blue themed user interface.

## Features
- Basic operations: addition, subtraction, multiplication, and division
- Simple and user-friendly interface
- Keyboard input support
- Clear and backspace functionality
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
index.html
README.md
LICENSE

## How to Run
1. Download or clone this repository
2. Open the index.html file in any web browser
3. Use the calculator
